Seguindo o conceito da [[Teoria dos Dígrafos|Teoria dos Dígrafos]], os [[Dígrafo Consonantal]], tem como principal caracteristica a dominância das consoantes.

| LH      | Lhama     |
| NH      | Banha     |
| SC      | Desculpa  |
| SÇ      | Nasça     |
| XC      | Exceto    |
| XS      | Exsudar   |
| RR      | Carro     |
| SS      | Passo     |
| QU      | Queijo    |
| GU      | Guilherme |
| XC      | Exceção   |
